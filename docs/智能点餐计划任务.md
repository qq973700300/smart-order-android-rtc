## 六、M5 逐项明细（38 项 · 112h）

### 5.1 设计（M5-01～04，各 2h）

**M5-01 离线边界**

| 能力 | 在线 | 离线 |
|------|------|------|
| 语音 RTC 对话 | ✓ | ✗（仅提示） |

| 支付 | 按业务 | 不做 |

**M5-02 ER 表**

| 表 | 字段要点 |
|----|----------|
| sync_queue | id, clientOpId, payloadJson, status, retryCount, createdAt |
| sync_log | queueId, httpCode, response, syncedAt |
| cart_snapshot | dishName, qty, updatedAt |
| order_mapping | localOrderId, serverOrderNumber |

**M5-03 同步流程**：FIFO → 有网 → 逐条 POST → 成功 DELETE → 失败 retryCount++ → 超 maxRetry 标记 FAILED  
**M5-04 冲突**：以 clientOpId 幂等；服务端重复提交返回同一单号则视为成功
